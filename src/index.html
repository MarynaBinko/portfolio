  • Gallery

    This React application is an image search and display tool. It features a search bar where users can input a topic to fetch relevant images from an API. The images are displayed in a gallery format, and users can click on any image to view it in a modal. The app uses a "Load More" button to fetch additional images.   • TO-DO List

    This project is a dynamic to-do list application that interacts with the JSONPlaceholder API to fetch, add, update, and delete tasks. Built with modern JavaScript practices, the application utilizes async/await for handling asynchronous HTTP requests via the Fetch API. Users can add new tasks through a form submission, with each task displayed in a list format. Tasks can be marked as completed or deleted by interacting with checkboxes and delete buttons, respectively. The application ensures smooth updates by modifying the DOM based on user actions. Error handling is implemented to manage unsuccessful operations gracefully. Additionally, the project showcases modular code organization for improved readability and maintainability.
